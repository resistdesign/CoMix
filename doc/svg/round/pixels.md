# svg-round-pixels

### Round all `(c)x(1,2)`, `(c)y(1,2)`, `width` and `height` pixel coordinates, on the target SVG object and all of it's descendant objects, to a whole number.

### WORKS WITH: Rectangles, circle/ellipse centers and lines.
